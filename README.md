# Eight4 Media - GitHub Hosting Package

This folder is a static package ready for GitHub Pages.

## Deploy

1. Create a new GitHub repository.
2. Upload all files inside this folder to the repo root.
3. In GitHub: `Settings > Pages`.
4. Source: `Deploy from a branch`.
5. Branch: `main` and folder: `/ (root)`.
6. Save and wait for deployment.

## Important

- Update `https://your-domain.com` in:
  - `index.html` meta tags
  - `sitemap.xml`
  - `robots.txt`
- If using a custom domain, add a `CNAME` file with your domain.

## Contact Form Note

This static version does not send server email by default.
For live lead delivery on static hosting, connect the form to a provider like Formspree, Getform, or Basin.
